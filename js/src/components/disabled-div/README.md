# DisabledDiv

Make the child component appear disabled by using CSS filter.

## Usage

```jsx
<DisabledDiv disabled={ disabled }>
  <MyComponent>
</DisabledDiv>
```

## API

| Property   | Description                               | Type    | Default |
| ---------- | ----------------------------------------- | ------- | ------- |
| `disabled` | Make the child component appear disabled. | Boolean | `true`  |
